# Tutorial de Instalação e Configuração

---

<div style={{ textAlign: "left", color:'white' }}>
  ## 1 - Instalação do servidor HTTP
  ### No linux instale o NGINX ou Apache com o seguinte comando: <br />

  <div style={{display:'flex'}}>
  <div style={{width:'100%'}}>
      #### Apache:
    Fedora/CentOS/Red Hat Enterprise Linux:
      <div
        style={{
          padding: "5px",
           background: "#3d3d3d",
          borderRadius: "6px",
          maxWidth: "80%",
        }}
      >
        sudo yum install httpd <br />
        sudo systemctl enable httpd <br />
        sudo systemctl start httpd
      </div>
      <br />
      Ubuntu/Debian:
      <div
        style={{
          padding: "5px",
          background: "#3d3d3d",
          borderRadius: "6px",
          maxWidth: "80%",
        }}
      >
        sudo apt install apache2 <br />
        sudo service apache2 start
      </div>
      </div>
   <hr style={{marginRight:'70px', border:'none',width:'1px',color:'#333', background:'gray'}} />
    <div style={{width:'100%'}}>
      #### NGINX:

      Fedora/CentOS/Red Hat Enterprise Linux:
      <div
        style={{
          padding: "5px",
           background: "#3d3d3d",
          borderRadius: "6px",
          maxWidth: "80%",
        }}
      >
        sudo yum install epel-release <br />
        sudo yum update <br />
        sudo yum install nginx
      </div>
      <br />
      Ubuntu/Debian:
      <div
        style={{
          padding: "5px",
          background: "#3d3d3d",
          borderRadius: "6px",
           maxWidth: "80%",
        }}
      >
        sudo apt-get update <br />
        sudo apt-get install nginx
      </div>

    </div>
    </div>
