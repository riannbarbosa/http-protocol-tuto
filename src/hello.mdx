# Tutorial de Instalação e Configuração

---
<div style={{ textAlign: "left" }}>
  ## Instalação do servidor HTTP com  Node.js

  Vamos utilizar o Node.js para o uso do protocolo HTTP

  ### Instalação
  1 - Clone o repositório 
 <div
        style={{
          padding: "5px",
           background: "#3d3d3d",
          maxWidth: "50%",
          color:'white',
        }}
      >
   ``` git clone https://github.com/alisonvor/redes-http ```
</div>

2 - Vá até o diretório
<div
        style={{
          padding: "5px",
           background: "#3d3d3d",
          maxWidth: "50%",
          color:'white',
        }}
      >
   ``` cd redes-http ```
</div>

 E instale as dependências executando <i><b>``` npm install ``` </b></i>.

 Para iniciar o servidor execute o seguinte comando:
 <div
        style={{
          padding: "5px",
           background: "#3d3d3d",
          maxWidth: "50%",
          color:'white',
        }}
      >
   ``` npm start ```
</div>
